Add featured slider elements to Mayflower sites.

== Description ==

This plugin adds a section to the sidebar of the Wordpress Admin Dashboard called "Featured Slider" to add featured slider elements to Mayflower sites.
Users will be able to view a list of slider posts and create, update, delete and re-order sliders. Upon creating a new slide, the user will be able to choose
a title, description, external link and featured image.

== Installation ==

1. Download plugin files from https://github.com/BellevueCollege/mayflower-homepage-carousel.
2. Place the plugin folder in your `wp-content/plugins/` directory.
3. Activate the plugin through the 'Plugins' menu in WordPress.
4. You will see the 'Featured Slider' add-on on the sidebar of the dashboard.
